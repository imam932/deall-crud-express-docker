# Skill Test - Deall BE
## Clone Project and run local
1. clone repository
```
git clone https://github.com/imam932/deall-crud-express-docker.git
```
2. open directory project
```
cd deall-crud-express-docker
```
3. start mongoDB service
```
sudo systemctl start mongod
```
3. run command
```
npm install
node service.js
```
